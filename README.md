# K-Culture Explorer 🏛️

한국 문화유산을 탐험하고, 탐방 후기를 나누고, AI가 설계한 나만의 여행 루트를 만나보세요.

## 🌟 기능

- **문화유산 탐색** - 12개 한국 문화유산을 카테고리별로 탐색
- **탐방 후기** - 여행자들의 생생한 탐방 경험 공유
- **AI 루트 플래너** - AI가 최적의 여행 루트와 준비사항 안내

## 🛠️ 기술 스택

- HTML5 / CSS3 / Vanilla JavaScript
- Apple-inspired Premium Dark Design
- Marked.js (마크다운 렌더링)

## 🚀 로컬 실행

```bash
python -m http.server 3000
# http://localhost:3000 접속
```
